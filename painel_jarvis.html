Sincronização entre dispositivos
GitHub — Sincronização automática Recomendado

Cole um Personal Access Token do GitHub com permissão contents:write no repositório. Qualquer dispositivo lê automaticamente ao abrir o painel; o admin escreve ao anexar bases.

Como criar o Personal Access Token
  1. Acesse github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Clique em "Generate new token"
  3. Defina expiração e em Repository access selecione apenas este repositório
  4. Em Permissions → Contents selecione Read and write
  5. Gere o token, copie e cole no campo acima
