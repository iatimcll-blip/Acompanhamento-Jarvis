📝 Instruções de Uso

Cada módulo acima possui um upload independente:

1. Painel de Redes → Upload de BASE_REDE.xlsx (BACKBONE + ACESSO)

2. B2B Abertos → Upload de BASE_ABERTOS.xlsx

3. B2B Encerrados → Upload de BASE_B2B_ENCERRADOS.xlsx

4. Repetidos → Usa dados de Encerrados (mesmo upload)

5. OFS → Upload de BASE_PRODUTIVIDADE_OFS.xlsx

6. Frota → Upload de base de frota quando disponível


✓ Cada base pode ser atualizada independentemente
✓ Dados persistem na sessão (até fechar o navegador)
✓ Gráficos atualizam automaticamente ao upload
